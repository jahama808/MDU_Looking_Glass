DEPLOYMENT IN 3 STEPS:
================================================================================

1. EXTRACT
   --------
   Linux/Mac:  unzip property-outage-dashboard.zip
   Windows:    Right-click > Extract All

2. INSTALL
   --------
   Linux/Mac:  chmod +x install.sh && ./install.sh
   Windows:    Double-click install.bat

3. START
   --------
   Linux/Mac:  ./start-all.sh
   Windows:    Double-click start-all.bat

   Access: http://localhost:5173

DATA PROCESSING:
================================================================================

Place your CSV files in the project directory:
- wan_connectivity-YYYY-MM-DD.csv
- Eero Discovery Details - YYYY-MM-DD HHMMSS.csv

Then run:
  source venv/bin/activate  (Linux/Mac)
  call venv\Scripts\activate.bat  (Windows)

  python process_property_outages_db.py \
    --connectivity-file wan_connectivity-2025-11-06.csv \
    --discovery-file "Eero Discovery Details - 2025-11-04 081045.csv"
